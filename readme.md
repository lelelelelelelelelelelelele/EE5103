# EE5103 课程资料归档（已完结）

本仓库用于整理与归档 **EE5103** 课程相关资料，目前已完结，主要用于给学弟学妹提供参考。

## 课程简介

EE5103（Computer Control Systems）是 NUS ECE 的研究生模块，聚焦计算机/微处理器控制系统与离散时间控制的基本概念、分析与设计方法，并涵盖部分工程实现要点（如采样、混叠、离散化误差等）。
不同学期的授课安排与考核可能调整，请以当学期 syllabus/Canvas 为准；本仓库主要整理 CA、cheatsheet 与往年试题等复习参考资料。
